2. Site Purpose

The purpose of Ki-Tribes is to document, preserve, and present educational information about the tribes and languages of the DRC. The site will include historical background, locations, cultural customs, and linguistic details. It serves as a resource for students, researchers, tourists, and anyone interested in learning more about the cultural diversity of the Congo.

8. JavaScript Functionality Plan

JavaScript will be used on the Ki-Tribes site to improve user interaction and navigation.

1. Mobile Navigation Toggle

Opens/closes the mobile menu when the user taps the menu icon.

function toggleMenu() {
    const nav = document.getElementById("mobile-nav");
    nav.classList.toggle("open");
}

2. Tribe Search Filter

Allows users to search tribes by name and filter results dynamically.

function searchTribes() {
    const input = document.getElementById("searchInput").value.toLowerCase();
    const cards = document.getElementsByClassName("tribe-card");

    for (let i = 0; i < cards.length; i++) {
        let name = cards[i].innerText.toLowerCase();
        cards[i].style.display = name.includes(input) ? "" : "none";
    }
}

3. Smooth Scrolling

Gives the site a smooth scroll effect when navigating between sections.

document.querySelectorAll('a[href^="#"]').forEach(link => {
    link.addEventListener("click", function(e) {
        e.preventDefault();
        document.querySelector(this.getAttribute("href")).scrollIntoView({
            behavior: "smooth"
        });
    });
});
