KiConnect — Site Plan

Dating platform for users across the Democratic Republic of Congo • Document uses chosen color scheme & Poppins font

1. Site Name

KiConnect

Reason for selection: “KiConnect” is short, memorable, and friendly. The syllable “Ki” suggests kinship and connection across many Congolese languages. The name communicates the core purpose — bringing people together for dating and relationships across all provinces of the DRC.

2. Site Purpose

KiConnect is a dating website tailored to people living in the Democratic Republic of Congo. The site provides:

  • Account creation and secure login for each user.
  • Personal profile pages with biography fields, age, province, and interests.
  • Profile picture upload and a multi-photo gallery per user.
  • Discovery/browsing of profiles (nearby provinces and nationwide).
  • Like / Match / Chat flow to begin conversations after mutual interest.

This purpose drives the site content, navigation, and feature set.

3. Scenarios

  • “How do I create an account and upload a profile picture so people can find me?” — The site will provide a clear sign-up form, file uploader for profile image, and a profile preview.
  • “How can I find matches from my province or search across other provinces?” — Filters and discovery options will let users search by province, age range, and interests.
  • “If someone likes me back, how will I be notified and start a conversation?” — The app will show matches in a Matches area and enable starting a message thread for matched users.

4. Color Scheme

At least two colors are defined below and where they will be used on the site.

#2E86AB
Primary — headings, primary buttons, icons
#F4D35E
Secondary — highlights, badges, callouts
#F0F0F0
Background — page background, large surfaces

Where colors are applied (examples):

  • --primary (#2E86AB): main headings, primary action buttons (e.g., “Like”, “Send Message”), navigation active state, icons.
  • --secondary (#F4D35E): secondary buttons, badge highlights (e.g., “New”, “Verified”), subtle section accents.
  • --bg (#F0F0F0): page background (used for the site plan document and the live site background).

This document itself uses these colors in headings, the logo gradient, swatches, and background to meet the rubric requirement.

5. Typography

Font selected: Poppins (Google Font).

How fonts are applied:

  • Poppins 600/700: used for headings, site name, and buttons to provide clear visual hierarchy.
  • Poppins 300/400: used for body copy, form labels, and small UI text for readability.

The Poppins font is imported and actively used in this HTML file (check the page styles to confirm).
